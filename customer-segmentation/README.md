# 🛍️ Customer Segmentation using K-Means

I used K-Means clustering to segment customers based on income and spending patterns. This helps businesses target different customer groups effectively

## 📌 Overview

This project segments customers based on purchasing behavior using K-Means clustering to enable targeted marketing strategies.

## ⚙️ Tech Stack

Python, Pandas, Matplotlib, Seaborn, Scikit-learn

## 🔍 Approach

* Data preprocessing and encoding
* Feature selection (Income & Spending Score)
* Elbow method to find optimal clusters
* K-Means clustering

## 📊 Key Insights

* Identified 5 distinct customer segments
* High-income high-spending customers are premium targets
* High-income low-spending customers need engagement strategies
* Low-income segments show different spending behaviors
