# 🏠 House Price Prediction Web App

## 📌 Overview

This project is a machine learning web application built using **Streamlit** to predict house prices based on user input features.

## ⚙️ Tech Stack

- Python
- Scikit-learn
- Streamlit

## 🚀 Features

- User-friendly interface for input
- Real-time predictions
- Integrated ML model

## ▶️ How to Run

```bash
pip install -r requirements.txt
streamlit run src/app.py
```

## 📊 Input Features

- Avg. Area Income
- Avg. Area House Age
- Avg. Area Number of Rooms
- Avg. Area Number of Bedrooms
- Area Population
